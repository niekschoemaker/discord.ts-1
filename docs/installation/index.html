discord.ts (discordx)

Create a discord bot with TypeScript and Decorators!

Introduction#

This module is an extension of discord.js, so the internal behavior (methods, properties, ...) is the same.

This library allows you to use TypeScript decorators on discord.js, it simplify your code and improve the readability !

Requirements#

NameVersion
Node.js>= 16.6.0
TypeScript>= 4.4.3

Easy setup - starter project#

  1. Clone this project
git clone https://github.com/oceanroleplay/discord.ts-examplecd discord.ts-example
  1. Build
  • npm
  • Yarn
npm install
yarn install
  1. Setup bot token
  • Windows
  • Linux
set BOT_TOKEN=<your bot token>
export BOT_TOKEN=<your bot token>
  1. Run
  • npm
  • Yarn
npm run start
yarn run start
  1. And you're done, everything has been done for you! ðŸš€

Installation#

  • Stable
  • Development

Use npm or yarn to install discordx with discord.js

caution

You also have to install "reflect-metadata" for the decorators

  • npm
  • Yarn
npm install discordx reflect-metadata discord.js
yarn add discordx reflect-metadata discord.js

Install your TypeScript dev dependencies too

  • npm
  • Yarn
npm install --save-dev @types/node typescript
yarn add --dev @types/node typescript

And you should see this in your package.json

package.json
{  // ...  "dependencies": {    "discordx": "^6.X.X",    "discord.js": "^13.X.X",    "reflect-metadata": "^0.1.13"  },  "devDependencies": {    "@types/node": "^16.X.X",    "typescript": "^4.X.X"  }  // ...}

If you need to update again in the future just run:

  • npm
  • Yarn
npm install discordx discord.js
yarn add discordx discord.js

Use npm or yarn to install discordx with discord.js

caution

You also have to install "reflect-metadata" for the decorators

  • npm
  • Yarn
npm install discordx@dev discord.js@dev reflect-metadata
yarn add discordx@dev discord.js@dev reflect-metadata

Install your TypeScript dev dependencies too

  • npm
  • Yarn
npm install --save-dev @types/node typescript
yarn add --dev @types/node typescript

And you should see this in your package.json

package.json
{  // ...  "dependencies": {    "discordx": "^7.X.X-dev.###",    "discord.js": "^13.X.X-dev.###.###",    "reflect-metadata": "^0.1.13"  },  "devDependencies": {    "@types/node": "^16.X.X",    "typescript": "^4.X.X"  }  // ...}

If you need to update again in the future just run:

  • Yarn

yarn add discordx@dev discord.js@dev

Execution environment#

To start your bot you can compile your code into JavaScript with TypeScript using the tsc command or simple use ts-node.

caution

Be aware that if you compile your code into JavaScript with tsc you have to specify .js files when you instanciate your Client

const client = new Client({  botId: "test",  // prefix: "!",  prefix: async (message: Message) => {    return "!";  },  classes: [    // glob string to load the classes. If you compile your bot, the file extension will be .js    `${__dirname}/discords/*.{js,ts}`,  ],  intents: [    Intents.FLAGS.GUILDS,    Intents.FLAGS.GUILD_MESSAGES,    Intents.FLAGS.GUILD_MEMBERS,  ],  silent: false,});
client.on("ready", async () => {  console.log(">> Bot started");
  // to create/update/delete discord application commands  await client.initApplicationCommands();  await client.initApplicationPermissions();});
client.login(BOT_TOKEN);

tsconfig.json#

Your tsconfig.json file should look like this:

tsconfig.json
{  "compilerOptions": {    "module": "commonjs",    "target": "ES2021",    "noImplicitAny": true,    "sourceMap": true,    "strict": true,    "outDir": "build",    "emitDecoratorMetadata": true, // required    "experimentalDecorators": true, // required    "declaration": false,    "importHelpers": true, // required    "forceConsistentCasingInFileNames": true,    "lib": ["ES2021", "esnext.asynciterable"],    "moduleResolution": "node"  },  "exclude": ["node_modules"]}

reflect-metadata#

caution

You should import the reflect-metadata module in your main file, for reflection. Otherwise, decorators may not function properly.

import "reflect-metadata";import { Client } from "discordx";
async function start() {  // ...}
start();
