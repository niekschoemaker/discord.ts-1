@ContextMenu

create discord context menu options with ease!

Here are some example screenshots:

Example#

@Discord()export abstract class contextTest {  @ContextMenu("MESSAGE", "message context")  async messageHandler(interaction: ContextMenuInteraction) {    console.log("I am message");  }
  @ContextMenu("USER", "user context")  async userHandler(interaction: ContextMenuInteraction) {    console.log("I am user");  }}

Signature#

ContextMenu(  type: "USER" | "MESSAGE",  name?: string,  params?: ApplicationCommandParams)

It will also disable the command from being usable in DMs."
